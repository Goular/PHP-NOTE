# ECSHOP3.0学习

### 安装
<pre>
	配置虚拟主机，配置域名，添加数据库，重启服务器，就可以使用了.
	管理员账号:admin,密码:admin123456
	安装完后会建议删除文件夹，我们就直接删除文件夹，这样会比较好.
</pre>

### 非代码使用笔记
<pre>
	商店设置--网店设置
		商店名称:	ECSHOP			[电子商城最下方显示的商店名称]
		商店标题:	ECSHOP演示站	[HTML标题栏]
		商店关键字:	ECSHOP演示站	[HTML关键字keywords]
		商店描述:	ECSHOP演示站	[HTML关键字keywords]
		
		商店logo：	需求的是logo.gif,但是实际显示logo.jpg,所以我们需要改变图片为logo.jpg

	商品分类中也存在关键字和描述，对分类的编辑中就可以设置分类关键字和分类描述，设置完成后会出现在HTML的meta中的keyswords上	

	商店设置--基本设置
		URL重写:	禁用  简单重写  复杂重写 
		URL重写是一种搜索引擎优化技术，可以将动态的地址模拟成静态的HTML文件。需要Apache的支持。
		
		一般的按照选中的时候，ecshop的提示去做就可以了，但是有一个问题就是使用xampp的时候，.htaccess需要添加一句:Options +FollowSymLinks
		
		禁用:
		http://www.ec-shop.com/goods.php?id=72
		简单重写效果:
		http://www.ec-shop.com/goods-72.html
		复杂重写效果：
		http://www.ec-shop.com/goods-72-智能相机.html
		
	商品管理--商品分类	
		首页右侧栏的显示，尽可显示指定个数，多的会隐藏，可以通过提高权重来显示，在分类中的编辑就可以进行处理
		
	系统设置--自定义导航栏
		可以通过权重来进行显示的控制
</pre>
